# Landing Page Pagar Nusa Ranting Balongmojo

Landing Page Pagar Nusa Ranting Balongmojo merupakan website company profile yang dibangun menggunakan **HTML5**, **CSS3**, dan **JavaScript**. Website ini bertujuan untuk memperkenalkan profil organisasi kepada masyarakat melalui tampilan yang modern, responsif, serta mudah diakses dari berbagai perangkat.

## ✨ Fitur

- Responsive Design
- Modern User Interface (UI)
- Mobile Friendly
- Interactive Navigation
- Fast Loading
- Clean Code Structure

## 📄 Halaman Website

- Home
- Sejarah
- Visi dan Misi
- Prestasi
- Galeri
- Tentang Kami

## 🛠️ Teknologi

- HTML5
- CSS3
- JavaScript

## 🎯 Tujuan Proyek

Website ini dikembangkan sebagai media informasi dan company profile Pagar Nusa Ranting Balongmojo. Melalui website ini, pengunjung dapat mengenal sejarah organisasi, visi dan misi, prestasi yang telah diraih, dokumentasi kegiatan, serta informasi mengenai organisasi secara lebih mudah.

## 🚀 Cara Menjalankan

1. Clone repository.

```bash
git clone https://github.com/username/landing-page-pagarnusa-balongmojo.git
```

2. Masuk ke folder proyek.

```bash
cd landing-page-pagarnusa-balongmojo
```

3. Buka file `index.html` menggunakan browser.
